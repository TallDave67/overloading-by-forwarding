# Overloading by Forwarding

This project demonstrates forwarding in C++20.

Linux distro "Pop!_OS 20.10"

cmake version 3.16.3

Ubuntu clang version 11.0.0-2

## Steps

cd build

./doCmake.sh

make VERBOSE=1

./overloading-by-forwarding
